" El desarrollo web front end es una de las dos caras de una misma moneda dentro de la creación de una web o del diseño de aplicaciones.
Junto al front end se encuentra el back end, cada una centrada en unos aspectos concretos. En este caso, el front end se encarga de la creación y el mantenimiento de la interfaz de usuario.

"El uso de aplicaciones y páginas web es una constante en nuestras vidas. De hecho, nos pasamos horas cada día navegando e interactuando con ellas.
Por eso, una de las profesiones más cotizadas y con mayores perspectivas de futuro es la del desarrollador front end, que es el perfil profesional que se encarga de crear las interfaces web."
